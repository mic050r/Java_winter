# <메모장 만들기>

HDD에 있는 방학메시지.txt → ImputStrem(1byte) ~~~Reader(2byte) → 메모장

메모장 → Output Stream → HDD

BufferedReader

buffer라는 창고같은 저장공간을 만든다. 대기장소

## <개선하기>

Format(JMenu) : 상위메뉴

Format의 메뉴 항목 : 글꼴크기, 글꼴모양

Format의 하위메뉴 : 글꼴색

글꼴색의 메뉴항목(JMenu) : 빨강, 파랑, 검정

나머지는 다 JMenuItem

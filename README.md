#< 성적프로그램 >

조건
- Swing패키지의 컨테인와 컴포넌트를 사용
- 한 학생에 대한 학생 정보 및 성적 정보 → 학번, 성명, 학년, 반, 국어, 영어, 수학, 과학, 총점, 평균
- 입력 : JTextField를 사용하여 입력 받는다.
- 입력받은 정보를 기번으로 2번의 내용을 JTable에 출력
- MVC model 사용여부는 선택
